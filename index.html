Interactive Elements

In-text link

CSS selector:

.link-ex

Sample code:

<a class="link-ex" href="https://ca.linkedin.com/in/astevenjones">Steven Jones</a>

Rendered element:

Steven Jones

Navigation Link

CSS selector:

.main-nav

Sample code:

<a class="main-nav" href="#citations">Citations</a>

Rendered element:

Citations

Submit/Reset Buttons

CSS selector:

Rendered element:
